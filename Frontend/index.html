Mario Trivia

Mario Trivia is a webapp that allows a user to play a 5 question trivia game on the subject of Mario. I utilized Asyncronus JavaScript to keep track of time by decrementing a number inside an object. Score is checked when a user clicks finish. If the value of selected option is equal to correct value, then correctScore is incremented. If it is not equal, then the incorrect score is incremented. At the end of the game all values are set back to original.

Challenge 1 - Trying to follow the logic of the game in JavaScript vs real life was mind bending. I was able to discover how to "think like a coder" from this project.

Challenge 2 - Learning setInterval and setTimeout led me down a path of discovering Asynchronus JavaScript. It was weird to understand and harder to implement, but by the end of it I even understood how the JS Call Stack works.

Challenge 3 - Keeping track of the score was interesting because I didn't really understand how the number was going to be there throughout the game. From this curiosity I learned more about memory and how JavaScript runs in the browser.
