Mario Trivia

  • HTML
  • CSS
  • JavaScript

Mario Trivia is a webapp that allows a user to play a 5 question trivia game on the subject of Mario. I utilized Asyncronus JavaScript to display a timer and determine transitions between questions. I kept track of time by incrementing a number inside an object which is stored in local memory. At the end of the game all values set back to original.

  • Challenge 1 - Trying to follow the logic of the game in JavaScript vs real life was mind bending. I was able to discover how to "think like a coder" from this project.
  • Challenge 2 - It took me a long time to be able to click start and let it count down to zero, show the right answer, then move to the next question start the cycle over again.
  • Challenge 3 - Keeping track of the score was interesting because I didn't really undersstand how the number was going to be there throughout the game. From this curiosity I learned more about memory and how JavaScript runs in the browser.
